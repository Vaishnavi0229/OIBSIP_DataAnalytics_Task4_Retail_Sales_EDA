# Retail Sales Exploratory Data Analysis (EDA)

## Objective
The objective of this project is to perform exploratory data analysis on retail sales data to identify patterns, trends, and business insights that can support data-driven decision-making.

## Dataset
- Retail Sales dataset containing transaction details, product categories, customer information, and sales amounts.

## Steps Performed
- Loaded and cleaned retail sales datasets
- Performed descriptive statistical analysis
- Conducted time-series analysis to identify sales trends and seasonal patterns
- Analyzed customer demographics and product performance
- Created visualizations including bar charts, line plots, and heatmaps to present insights

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Outcome / Results
- Identified top-performing product categories contributing major revenue
- Observed seasonal variations affecting sales performance
- Generated actionable insights useful for inventory planning and targeted marketing strategies
